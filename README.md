# 🔊 Edge Audio Recognition
본 프로젝트는 저사양 엣지 디바이스(키링 형태)에서 실시간 소리 분석을 수행하기 위한 경량화 딥러닝 모델 실험 저장소입니다. 화재 경보와 같은 긴급 위험 상황뿐만 아니라, 일상적인 생활 소음까지 감지하여 사용자에게 시각 및 촉각적 피드백을 제공하는 것을 목표로 합니다.

## 🎯 Project Goals
High Accuracy: 생활 소음 및 위험 상황에 대해 80% 이상의 분석 정확도 확보.

Model Optimization: 엣지 디바이스 탑재를 위한 모델 경량화(TinyML) 및 파라미터 최소화.

Context Awareness: 단순 위험 감지를 넘어, 파인튜닝을 통해 다양한 생활 소음(도어락, 아기 울음, 벨소리 등) 인지.

Real-time Processing: 1~2초 단위의 실시간 오디오 스트림을 연속적으로 처리하는 저지연 파이프라인 구축.
